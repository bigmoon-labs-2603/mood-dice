# Mood Dice 🎲

A tiny fun web app that suggests something interesting to do based on your current mood.

## ✨ Features

- Choose a mood (Happy / Bored / Stressed / Tired / Curious)
- Roll random suggestions instantly
- Re-roll for new ideas
- Pure HTML/CSS/JS — no dependencies

## 🚀 Live Demo

Open `index.html` directly in your browser.

## 🧩 Tech Stack

- HTML
- CSS
- Vanilla JavaScript

## 📁 Project Structure

- `index.html` — the main app
- `README.md` — project description

## 🛠️ Quick Start

1. Clone this repo
2. Open `index.html` in browser
3. Pick a mood and roll 🎲
